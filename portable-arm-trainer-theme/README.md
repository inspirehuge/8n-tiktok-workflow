## 🚀 Installation

1. **Download** the theme ZIP file
2. **Upload** to your Shopify admin:
   - Go to Online Store > Themes
   - Click "Upload theme"
   - Select the ZIP file
3. **Activate** the theme
4. **Customize** via the theme editor
